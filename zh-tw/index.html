Codex Meetup Tokyo #1 回顧

我在 Codex Meetup Tokyo #1 進行了閃電演講。 正如標題「回顧」所暗示的，想必大家已經猜到內容如何了。 昨日はCodex Meetup Tokyo #1 で登壇させていただきまして本当にありがとうございます。 今回は内容面でも声の調子でも課題が残り、多くの反省を得る登壇となりました。… — びーぐる / Beagle Works (@beagle_dog_inu) March 20, 2026 （摘錄部分省略的內容） 非常抱歉讓大家聽到如此不流暢的演講。我保證會將這次的反省充分活用（如果還有下次的話⋯⋯），以最佳狀態和充分準備登上講台。 這次也與許多人交流，學到了很多。真的非常感謝。 反省點1 - 投影片的過度堆砌 演講前幾天身體不適，喉嚨和聲音出了問題。我決定改變策略，將所有該說的內容都塞進投影片，即使無法順利說話也能靠資料撐過去。 結果導致整場演講的完成度下降。除了在投影片中塞入過多原本該口述的內容外，由於持續編輯到最後一刻，未能充分整理資料的架構。 反省點2 - 聲音問題 我原以為到了當天喉嚨已經恢復得差不多了，但到了現場還是出現了難以發聲的症狀。 演講中無法流暢地說話，加上反省點1的問題，整場演講變得非常難以聽取。 反省點3 - 題材選擇 不能迴避題材選擇不當的問題。我將原本為15分鐘講座準備的題材濃縮成5分鐘閃電演講的內容。結果論點變得難以理解，成了一場無法傳達重點的演講。在確定為閃電演講時，就應該與主辦方商量更換題材。 更根本的問題是，試圖將規格驅動開發帶入15分鐘講座，說得客氣點，簡直是瘋了。 我需要重新冷靜思考：重要的不是我想講什麼題目，而是聽眾需要什麼題目。我應該重新審視自己在社群中的角色。 反省點4 - 知識不足 乍看之下這似乎與演講內容沒有直接關係，但我深感在最新資訊的吸收以及 Codex 相關知識方面都明顯不足。 題材選擇不當也有知識不足的因素。 「反正問 AI 就知道了」的自滿心態，以及因為忙於演講和社群活動而忽略了作為軟體工程師持續學習技術這一本分——這些都是需要深刻反省的。 做得好的地方 關於使用 Codex 的規格驅動開發，感覺部分與會者對我的主張有了一定程度的理解和認同。 投影片的內容獲得了一定程度的好評 總結 以下事項必須徹底執行： 充分準備並熟記投影片。記住每一項內容的撰寫意圖並加以傳達 根據活動宗旨和聽眾選擇題材。優先考慮被需要的內容，而非自己想講的內容 做好健康管理 我由衷感謝獲得上台演講的機會，以及當天與我交流的每一位。 如果有下次機會能將這次的反省付諸實踐，我將全力準備以呈現一場好的演講。

讀完《Claude Code AI驅動開發入門》

讀完了《Claude Code AI驅動開發入門》（Claude Code によるAI駆動開発入門）。 部分感想已發布在X上。 池袋に行ったのでジュンク堂でClaude CodeによるAI駆動開発入門、買いました。これはもういい意味で教科書ですね。使ったことないひとは体系的に学べる、使ったことあるひとはパラパラめくるだけで何かしら発見があるようなイメージ。 pic.twitter.com/fUPNrjwPQz — びーぐる (@beagle_dog_inu) December 10, 2025 既にほとんど知っている事柄に関して、他の人の異なるアプローチや考え方にふれる、記事や書籍を読む、ってのは本当によいことで。 新たな知識は増えないけど、代わりに新たなインスピレーションや気付きは発生するのでとても大事にしている。当然理解も深まり言語化もうまくなる。 — びーぐる (@beagle_dog_inu) December 20, 2025 前幾天參加Claude Code Meetup Tokyo時，也向多位朋友分享了這本書的優點。 Claude Code Meetup Tokyo · Luma https://luma.com/ioed4ksj?tk=U0Wvfo 這本書除了以教科書般的方式涵蓋Claude Code的基礎之外，最大的價值在於能看到作者平川先生在面對特定需求時如何指示Claude Code、了解「他在想什麼」。即使是已經很熟悉Claude Code的人，從這個角度來閱讀這本書也會收穫良多。我自己也學到了很多。 而且，我竟然有幸直接見到了作者平川先生本人並與他交流！其中令我印象特別深刻的是，他將自己的著作比喻為DUO 3.0。 DUO 3.0是日本著名的英語單字和片語學習書。雖然可以當作「單字本」使用，但它的例句非常優秀——透過學習例句，可以一次掌握單字的意義、用法、慣用語和固定表達。換句話說，這本書的結構讓你能非常高效地學習英語。 我覺得《Claude Code AI驅動開發入門》確實實現了類似DUO 3.0的學習效率。它不只是命令參考手冊或技巧集——你可以看到作者在處理實際任務時寫了什麼樣的提示詞、如何與Claude Code互動。透過這些實例，自然而然就能學會提示詞的撰寫方式和思考過程。 就像DUO 3.0不是「讀一次就結束」的書，而是放在手邊反覆參閱的書一樣，這本書也是在持續使用Claude Code的過程中反覆參考，逐漸磨練自己技能的風格⋯⋯它真正具有作為「Claude Code開發實踐教科書」的價值。 無論是即將開始使用Claude Code的人，還是已經在使用的人，都能從中獲得新的發現，非常推薦。 Claude Code によるAI駆動開発入門 Claude Codeの基本から活用方法までこの一冊でぜんぶ学べる！ Amazon.co.jp
